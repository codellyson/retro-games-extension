# Privacy Policy

**Retro New Tab** does not collect, transmit, or sell any personal data.

## What the extension stores

The extension uses `chrome.storage.sync` to save the following on your own Chrome profile:

- Game high scores
- Last-played game
- Theme choice (Glassy / Neon / Paper)
- Difficulty (Easy / Normal / Hard)
- Sound muted / unmuted
- Whether the bookmarks bar is shown

If you're signed in to Chrome, this data syncs across your devices through Chrome itself. The developer has no access to it at any point.

## Search box

The new-tab page includes a search box. When you submit a query, your browser navigates directly to Google. The query goes from your browser to Google and is subject to [Google's privacy policy](https://policies.google.com/privacy). The extension doesn't see or log queries.

## What's not here

- No analytics
- No trackers
- No third-party scripts
- No remote code execution (all JavaScript is bundled in the extension package)
- No ads
- No network requests of any kind from the extension itself

## Changes

If this policy ever changes, the updated version will live at the same URL as this document. Material changes will be called out in the commit history of the repository.
